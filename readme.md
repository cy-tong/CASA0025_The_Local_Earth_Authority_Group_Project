# Mapping London Thermal Vulnerability

This repository contains the source code for the **London Thermal Vulnerability** project developed for the CASA0025 module by The Local Earth Authority group. 

The project evaluates spatial inequalities in thermal exposure across London by mapping extreme summer heat, extreme winter cold and spatial access to public protective infrastructure (cool spaces and warm banks).

### Repository Structure

* `index.qmd`: The primary Quarto markdown file containing the project methodology, literature review and code documentation.
* `GEE_thermal_vulnerability_app.js`: The underlying Google Earth Engine JavaScript application. It features a bespoke data dashboard with interactive tooltips and utilises precomputed hybrid assets to guarantee instant application load times.
* `docs/`: The compiled HTML website served securely via GitHub Pages.

### Application

The compiled Quarto website and interactive Earth Engine application can be viewed live via the GitHub Pages deployment URL for this repository.
